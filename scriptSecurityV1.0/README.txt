=== scriptSecurity Module ===

Modul scriptSecurity adalah modul keamanan skrip yang dirancang untuk memastikan persetujuan dan integritas skrip. Modul ini dibuat oleh zaenishi dan dapat digunakan untuk mengamankan skrip Anda dari penggunaan yang tidak sah.

=== Penggunaan ===

1. Instalasi:
   Untuk menggunakan modul ini, pastikan Anda telah menginstal Node.js di komputer Anda. Kemudian, buat folder proyek dan salin file `scriptSecurity.js`, `config.js`, dan `package.json` ke dalamnya. Setelah itu, jalankan perintah `npm install` untuk menginstal dependensi yang diperlukan.

2. Penggunaan di dalam script:
   - Impor modul `scriptSecurity` di dalam script Anda.
   - Gunakan fungsi-fungsi yang disediakan oleh modul tersebut untuk memeriksa persetujuan, menyetujui skrip, memvalidasi data persetujuan, memeriksa integritas skrip, dan lain-lain.
   - Sesuaikan path yang diperlukan dalam file `config.js` sesuai dengan struktur direktori proyek Anda.

3. Jalankan proyek:
   Anda dapat menjalankan proyek Anda menggunakan perintah `npm start`.
